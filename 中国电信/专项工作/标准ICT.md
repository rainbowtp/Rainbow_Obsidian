# 了解到的情况：
1. 交付
	1. 目前由综维统一负责；各区局的综维人员为一线交付，综维中心为二线交付。
	2. 交付只负责给设备通电通网，之后就可以回单。在公司考核激励的制度层面，并没有规范应用交付。 
2. 采购
	1. 高阶网关目前由公司按集采库里的目录统一采购
	2. 工作流程为：各区县单元举手采购，上报请求到商客部马江，马江汇总给云中台交付部鲁铖，鲁铖发起OA审批，经由各领导审批通过后进行采购。
	3. ==根据目前研发部二开的实际需要，建议公司将不同型号的机器（共四款）各采购一台用于研发测试。==
3. 试用
	1. 集团公司不对产品具体流程做要求，能否试用取决于市公司的流程设定。据了解，渭南公司是提供了试用流程的，建议西安公司也提供试用。 
4. 会议要求（2026.1.12）
	1. 华为：设备规格清单和全国成功案例
	2. 商客：15个合同的履约情况和问题；向省公司确认目前的四大类20多个场景。
	3. 技术能力侧：产品测试用例
	4. tip：需求、方案、成本、工期
